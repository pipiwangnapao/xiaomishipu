  食物苯丙氨酸查询 · 获得微信可打开的链接
  无需服务器，免费，约 5 分钟
========================================

本文件夹里有一个 index.html，把它放到「能通过网址访问」的免费空间，
就能得到一个 https 链接，在微信里打开即可使用。

任选下面一种方式即可（推荐方式一，最快）。

----------------------------------------
方式一：Netlify 拖拽上传（最简单）
----------------------------------------

1. 用电脑浏览器打开： https://app.netlify.com

2. 用邮箱或 GitHub 注册/登录（免费）

3. 在页面找到 "Want to deploy a new site?" 或「添加新站点」
   → 选择 "Deploy manually" / 「手动部署」或「拖放」

4. 把「本文件夹」整个拖进页面的虚线框里
   （拖「食物苯丙氨酸查询-可分享」这个文件夹，不要只拖 index.html）

5. 等几秒，会生成一个链接，类似：
   https://随机名字.netlify.app

6. 复制这个链接，发到微信（发给自己或好友），点开就能用。

提示：登录后可在 Netlify 里给站点改名字，链接会变成 https://你起的名字.netlify.app

----------------------------------------
方式二：GitHub Pages（适合会一点 Git 的人）
----------------------------------------

1. 注册 GitHub： https://github.com

2. 新建仓库（New repository）
   - 仓库名随便，例如： phe-query
   - 选 Public，创建

3. 进入仓库，点 "Add file" → "Upload files"
   - 把本文件夹里的 index.html 拖进去上传
   - 点 "Commit changes"

4. 点仓库顶部的 "Settings" → 左侧 "Pages"
   - Source 选 "Deploy from a branch"
   - Branch 选 main，文件夹选 / (root)，Save

5. 等 1～2 分钟，页面上会显示你的链接：
   https://你的用户名.github.io/仓库名/

6. 把这个链接发到微信，点开即可。

----------------------------------------
方式三：Vercel（和 Netlify 类似）
----------------------------------------

1. 打开 https://vercel.com 并登录（可用 GitHub）

2. 点 "Add New" → "Project"，选择 "Import Third-Party Git Repository"
   或直接上传：有的界面有 "Deploy" 或拖拽上传

3. 若没有上传入口：可先把本文件夹放到 GitHub 仓库，
   然后在 Vercel 里 "Import" 该仓库，自动部署后会给你链接。

----------------------------------------
常见问题
----------------------------------------

Q：微信里点链接打不开？
A：确保链接是 https 开头；若是 Netlify/GitHub 的链接在国内有时会慢，
   多等几秒或换个网络试试。

Q：可以换一个更好记的链接吗？
A：Netlify / Vercel 登录后可在站点设置里改子域名（如 xxx.netlify.app）。
   GitHub Pages 若用「用户名.github.io」仓库，链接可以是 https://用户名.github.io

Q：会收费吗？
A：以上三种在正常个人使用下都是免费的。
