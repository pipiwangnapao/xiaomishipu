========================================
  食物苯丙氨酸查询 - 可分享包（最新）
========================================

本文件夹可直接用于部署到 Netlify（Git 方式），或压缩后发给别人部署。

【必须保留的目录结构】
  本文件夹根目录/
    index.html          ← 网站首页，必须在这里
    netlify.toml        ← Netlify 配置
    netlify/
      functions/
        phe-estimate.js ← AI 估算云函数

【部署到 Netlify】
1. 把本文件夹里的「所有内容」上传到 GitHub 仓库的「根目录」：
   - 根目录下直接是 index.html、netlify 文件夹、netlify.toml 等，不要再多一层子文件夹。
2. 在 Netlify 连接该仓库，Base directory 留空，发布目录留空或填 "."。
3. 需要 AI 估算时：在 Netlify 后台添加环境变量，见「Netlify环境变量配置-详细步骤.txt」。

【若部署后首页 404】
  看「首页404-修复办法.txt」。

【若 AI 估算不生效】
  看「AI估算不生效-排查步骤.txt」。
