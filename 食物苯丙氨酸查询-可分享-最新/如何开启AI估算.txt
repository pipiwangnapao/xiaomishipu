========================================
  表中查不到时「用 AI 估算」功能说明
========================================

当食物在本地表格中查不到时，页面会显示「用 AI 估算」按钮，
点击后由大模型估算该食物每 100g 的苯丙氨酸含量。

支持多种大模型（可选用免费/低价方案）：
  • DeepSeek（推荐，有免费额度）
  • Kimi（Moonshot，有免费额度）
  • 豆包（火山引擎）
  • OpenAI（需付费）

----------------------------------------
一、环境变量配置（在 Netlify 后台）
----------------------------------------

1. 选择用哪个模型（必填其一）：

   LLM_PROVIDER = deepseek   （默认，推荐）
   或 kimi
   或 doubao
   或 openai

2. 按所选模型配置对应 API Key：

   【DeepSeek】免费额度大、价格低
   • DEEPSEEK_API_KEY = 你的 DeepSeek API Key
   • 获取：https://platform.deepseek.com 注册并创建 API Key

   【Kimi / Moonshot】
   • MOONSHOT_API_KEY = 你的 Kimi 开放平台 API Key
   • 获取：https://platform.moonshot.cn 控制台 → API Key 管理

   【豆包 / 火山引擎】
   • DOUBAO_API_KEY = 火山方舟 API Key
   • DOUBAO_ENDPOINT_ID = 推理接入点 ID（形如 ep-2024xxxxxx）
   • 获取：火山引擎控制台 → 火山方舟 → 模型推理 → 创建接入点

   【OpenAI】
   • OPENAI_API_KEY = 你的 OpenAI API Key

----------------------------------------
二、部署方式（AI 估算才能生效）
----------------------------------------

仅拖拽上传静态文件夹时，云函数不会部署，AI 估算不可用。
请任选其一：

方式 A：Netlify 连接 GitHub 仓库
  1. 把本文件夹推送到 GitHub 仓库
  2. Netlify → Add new site → Import from Git → 选该仓库
  3. 在 Site settings → Environment variables 里添加上述变量
  4. 部署

方式 B：Netlify CLI
  1. 安装：npm install -g netlify-cli
  2. 在本文件夹执行：netlify deploy --prod
  3. 在 Netlify 后台为该站点设置环境变量
  4. 再次 netlify deploy --prod

----------------------------------------
三、费用与免责
----------------------------------------

• DeepSeek / Kimi 等有免费额度，具体以各平台为准。
• 估算结果仅供参考，不作为医疗或营养决策依据。
